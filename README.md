# 🎮 Neon Tetris Game

A modern **Neon-themed Tetris Game** built using **React.js**, featuring smooth gameplay, vibrant UI, and classic block-stacking mechanics.

---

## 🚀 Features

- 🌈 Neon-style UI design  
- 🎮 Classic Tetris gameplay  
- ⏱️ Real-time block movement & rotation  
- 🔢 Score tracking system  
- ⚡ Fast and responsive controls  
- 🧱 Collision detection & line clearing  
- 📱 Responsive design (works on desktop & mobile)  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS / Tailwind (if used)  
- **State Management:** React Hooks (useState, useEffect)  
- **Game Logic:** JavaScript  

---

## Run Locally

**Prerequisites:**  Node.js

## ⚙️ Installation & Setup

1. Clone the repository  
   - bash / CMD 
   `git clone https://github.com/Sultan-08/Neon-Tetris-Game.git`
2. Move to dir
   `cd neon-tetris-game`
3. Install dependencies:
   `npm install`
4. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
5. Run the app:
   `npm run dev`
